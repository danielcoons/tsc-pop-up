|The input (DBL) popup will display information to the user and wait for the operator to enter a value and click the button.

**Note:** This popup type allows the user to define a custom image for the display. The developer should test all settings prior to deployment to make sure that all images and text fit on the screen correctly.

**Note:** The function **Close Pop-Up (DBL).vi** can be used to send a value when closing this dialog. Otherwise, use Close Pop-Up.vi to close it early.

Inputs: +
-Graphic [path] - path to the image file to display +
-Message [String] - String Variable to populate Display with +
-Theme [lvclass] - the theme class for the pop up +
-Default Input [DBL] - default value for the prompt +

Outputs: +
-InputValue [DBL] - the user's input data
