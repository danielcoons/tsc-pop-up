|Display Two Button Popup
|image:Pop_Up.lvlib_Two_Button.lvclass_Display_Two_Button_Popup.vi.png[Pop-Up.lvlib:Two Button.lvclass:Display Two Button Popup.vi]
|The two button popup will display information to the user and wait for the operator to choose a button.

**Note:** The function **Close Pop-Up (Two Button Choice).vi** can be used to close this dialog early and send a value for the choice of Boolean selection. Otherwise, to just close the dialog early, use Close Pop-Up.vi.

Inputs: +
-Graphic [typedef combobox] - Image (PNG) for Graphic for Popup +
-Message [String] - String Variable to populate Display +
-Theme [lvclass] - the theme for the style of the popup +
-YText [string] - the string for the YES button +
-NText [string] - the string for the NO button

Outputs: +
-Choice [boolean] - the option that was chosen by the user
