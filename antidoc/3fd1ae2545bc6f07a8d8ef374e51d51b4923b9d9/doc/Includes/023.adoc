|Display Three Button Popup (Plain)
|image:Pop_Up.lvlib_Plain___Three_Button.lvclass_Display_Three_Button_Popup_(Plain).vi.png[Pop-Up.lvlib:Plain - Three Button.lvclass:Display Three Button Popup (Plain).vi]
|The one button popup will display information to the user and wait for the operator to click the  button.

**Note:** The function **Close Pop-Up.vi** can be used to close this dialog early.

**Note:** The dialog vertical sizing will adjust for longer messages, so **test** all messages prior to deployment to make sure that scaling fits.

Inputs: +
-Dialog Title [String] - String variable to populate the dialog's title bar +
-Message [String] - String Variable to populate Display +
-Theme [lvclass] - the theme for the style of the popup +
-Action 0  [String] - the button text for the **Action 0** button (far right) +
-Action 1 [String] - the button text for the **Action 1** button (middle) +
-Action 2 [String] - the button text for the **Action 2** button (far left) +
-Keep transparent buttons? [Boolean] - choose to leave buttons transparent (T) or use theme colors (F)

Outputs: +
-Choice [U8] - user selection of button 0, 1, or 2

|This is the override method required for each child of the pop up class.  It is the pop up that will display to the user.

To extend this function, create a child class of the pop up class. Then create an override method for this VI. **Display.vi** is a protected function, therefore it cannot be called in a main application. This VI must be called by a caller function. See one of the standard pop-ups for a setup example.
