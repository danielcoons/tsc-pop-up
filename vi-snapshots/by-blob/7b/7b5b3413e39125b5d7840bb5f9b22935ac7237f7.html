Pop-Up.lvlib:Plain - Two Button.lvclass:Display Two Button Popup (Plain).vi

The two button popup will display information to the user and wait for the operator to click the button to make a choice.

<b>Note:</b> The function <b>Close Pop-Up.vi</b> can be used to close this dialog early.

<b>Note:</b> The dialog vertical sizing will adjust for longer messages, so <b>test</b> all messages prior to deployment to make sure that scaling fits.

Inputs: +

-Dialog Title [String] - String variable to populate the dialog's title bar +

-Message [String] - String Variable to populate Display +

-Theme [lvclass] - the theme for the style of the popup +

-Y-Text [String] - the button text for the <b>yes</b> button (right) +

-N-Text [String] - the button text for the <b>no</b> button (left) +

-Keep transparent buttons? [Boolean] - choose to leave buttons transparent (T) or use theme colors (F)

Outputs: +

-Choice [Boolean] - true/false of user selection

The rules for the dialog component from Material v3 define the dialog contents to have buttons that are transparent.

If true, this follows those rules, if false, it will use the theme data for the button colors.
