Pop-Up.lvlib:Timeout One Button.lvclass:Display Timeout One Button Popup.vi

The one button popup will display information to the user and wait for the operator to click the button or until the timeout expires.

<b>Note:</b> The function <b>Close Pop-Up.vi</b> can be used to close this dialog early.

Inputs: +

-Graphic [typedef combobox] - Image (PNG) for Graphic for Popup +

-Message [String] - String Variable to populate Display +

-Theme [lvclass] - the theme for the style of the popup +

-Timeout (s) [DBL] - the amount of seconds for the timeout +

-Button Text [String] - the boolean text of the button
